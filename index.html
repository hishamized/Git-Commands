Basic Git Commands

No Cheatsheets

You don't have to explicitly make cheatsheets to remenber basic git commands. Just bookmark this page and I will keep adding all the basic commands that we use in git bash on daily basis.

All you need is to remember the URL of this page and you can visit it anytime you need to look for a particular command. You may copy and paste git commands into your notepad from here directly.

Relevant Sequence

I will add git commands on this page in a useful sequence. The commands which are usually require in the beginning will be displayed first and the rest later. It makes everything so simple.

  • git config --global user.name " username " [ used to login inside gitbash with your usernamem ]
  • git config --global user.email " your_email " [ used to login inside gitbash with your email ]
  • git init [ Initializes a local git repository ]
  • git status [ Displays the current status of the local git repository ]
  • git add -A  [ Adds all the files and directories present in the local git repository into the staging area ]
  • git commit -m" Commit Message" [ registers a commit with a commit message and anchors that particular moment of your code into the timeline ] 
  • git remote add origin "URL of the github repository" [ creates a remote repository of the same origin which is linked to the repository on your github account ]
  • git remote [ displays the names of the remote repositories that you have created ]
  • git remote -v [ displays the names of the remote repositories along with the URLs associated with them ] 
  • git push origin master [ Pushes all the files from your local repository into the master branch of your github repository using the remote repository named origin ]
